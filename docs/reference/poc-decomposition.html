OPTION 2 · pre-baked layers

Isometric A — tilt baked in Figma

Layers already carry the isometric angle + their own soft shadows. I only slide them.
Cheapest to run. Motion is pure translate — the compositor's favourite job. Shadows are painted into the PNGs, so no live blur.  •  Maintenance: to change the angle, spacing, or a typo, you re-export from Figma. The look lives in the assets, not the code.
OPTION 3 · flat layers + CSS 3D

Flat A — tilt applied in code

Flat straight-on layers. I set the angle, depth, and lean in CSS — tune it forever without touching Figma.
Still GPU-friendly, slightly heavier: live perspective + 3D rotate and one CSS shadow per layer. The whole stack can lean toward the cursor.  •  Maintenance: the angle, depth, and motion are all code — I retune them anytime. You only re-export a flat piece when its content changes.
